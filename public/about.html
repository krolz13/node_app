About This DevOps Homelab

This project represents a complete, professional Git-to-Production DevOps pipeline running entirely self-hosted on a Proxmox VE server. By containerizing a Node.js web application and establishing automated GitLab CI/CD jobs, we ensure that every code commit is automatically vetted, built into a microservice image, and securely rolled out with zero downtime.

Proxmox Virtualization
Leverages Debian 12 Cloud-Init VMs for hypervisor-level isolation of infrastructure services. VM 200 handles the CI/CD orchestration, while VM 201 hosts production containers.
GitLab CI/CD Pipelines
Automated workflows running inside a Docker executor run ESLint for linting, Jest for unit testing, and Docker-in-Docker to build and push container images.
Production Dockerization
Uses multi-stage Docker builds to package the Express.js app inside a secure, alpine-based runner running under a non-privileged system user account.
